# Podokop Production Walkthrough

Production deployment walkthrough video showing all major screens of the Podokop application.

## Video

[Download Walkthrough Video](podokop-walkthrough.mp4)

## Screens Covered

1. **Login** - Admin authentication
2. **Dashboard** - Welcome screen with quick actions
3. **Nouvelle Prescription** - Prescription type selection
4. **Prescription Form** - Normal prescription form (Prescription d'orthèses)
5. **Prescriptions** - List of all prescriptions (45 total)
6. **Scans 3D** - 3D foot scan management (27 scans)
7. **User Management** - User administration panel

## Notes

The normal prescription form (screen 4) demonstrates the comprehensive input interface for podiatric orthotic prescriptions, including:
- Patient and clinic information
- Podiatrist details
- Mold modifications
- Cutting specifications
- Stabilizers (front & back)
- Materials and specifications
- And more...
